understanding tensors

tensors in machine learning:

tensors are multi-dimensional arrays that represent complex data structures. in ml, they're used to store and process data efficiently.

  • 0d tensor: a scalar (single number)
  • 1d tensor: a vector (list of numbers)
  • 2d tensor: a matrix (table of numbers)
  • 3d+ tensor: higher dimensional data structures

tensors allow for parallel processing and are crucial for handling large datasets in neural networks and deep learning models.
